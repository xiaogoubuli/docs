$ ls [选项]... [文件]...
列表信息
-a 显示隐藏文件
-l 显示长格式
-h 显示人性化大小
-d 显示目录信息
-i 显示inode号
示例：
$ ls -alh / /etc
$ ls -d /etc
$ ls -i /

$ mkdir [选项]... 目录...
创建新目录
-p 递归创建
示例：
$ mkdir -p /tmp/a/b/c
$ mkdir /tmp/a /tmp/b /tmp/c

$ cd [-L|-P] [目录]
切换工作目录
示例：
$ cd ..
$ cd -
$ cd
$ cd ~

$ pwd
显示当前工作路径

$ rmdir [选项]... 目录...
删除空目录

$ cp [选项]... 源... 目录
复制文件或目录
-r 复制目录
-p 保留文件属性
示例：
$ cp -rp /etc/src /tmp

$ mv [选项]... 源... 目录
剪切或重命名
示例：
$ mv /etc/src /tmp

$ clear 或快捷键 CTRL + L
清屏

$ rm [选项]... 文件...
删除文件
-r 删除目录
-f 强制删除
示例：
$ rm -rf /tmp/abc
$ rm -rf *

$ touch [选项]... 文件...
创建空文件，更新访问与修改时间
示例：
$ touch /tmp/test

$ cat [选项]... 文件...
显示文件内容
-n 显示行号
示例：
$ cat -n /etc/issue

$ tac [选项]... 文件...
反向列示文件内容
示例：
$ tac /etc/issue

$ more 文件
分页显示文件内容
空格或f 下一页
b       上一页
回车    下一行
q或Q    退出
示例：
$ more /etc/services

$ less 文件
同more，另有上下左右滚动、翻页及搜索等功能
示例：
$ less /etc/services

$ head [选项]... [文件]...
显示文件的前几行（默认前10行）
-n 指定行数
示例：
$ head -n 20 /etc/services

$ tail [选项]... [文件]...
显示文件的后几行（默认后10行）
-f 动态显示文件末尾内容
示例：
$tail -n 20 /etc/services
tail -f /var/log/messages

$ ln [选项]... 目标 链接名
-s 创建软链接
示例：
$ ln -s /etc/issue /tmp/issue.sotf
$ ln /etc/issue /tmp/issue.hard
	硬链接 同步更新，通过i节点识别，不能跨分区，不能对目录使用硬链接

$ chmod [选项]... 模式... 文件...
更改文件的权限
-R 递归
模式 [ugoa]*([-+=]([rwxXst]*|[ugo]))+
示例：
$ chmod u+x,g-w,o=rwx /tmp
$ chmod 644 /tmp
$ chmod -R 777 /tmp
  文件 目录
r 读   文件列表
w 写   创建删除文件
x 执行 进入目录

# chown [选项]... [所有者][:[所有组]] 文件...
改变文件的所有者或所属组
示例：
# chown root /tmp
# chown :root /tmp
# chown root: /tmp
# chown root:root /tmp

# chgrp [选项]... 所属组 文件...
改变文件的所属组
示例：
# chgrp root /tmp

$ umask [-p] [-S] [模式]
显示设置文件的缺省权限
-S 以rwx形式显示新文件的缺省权限
示例：
$ umask
0002
$ umask -S
$ umask 0022
