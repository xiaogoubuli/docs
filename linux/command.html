$ ls [选项]... [文件]...
-a 显示隐藏文件
-l 显示长格式
-h 显示人性化大小
-d 显示目录信息
-i 显示inode号
示例：
$ ls -alh / /etc
$ ls -d /etc
$ ls -i /

$ mkdir [选项]... 目录...
-p 递归创建
示例：
$ mkdir -p /tmp/a/b/c
$ mkdir /tmp/a /tmp/b /tmp/c

$ cd [-L|-P] [目录]
示例：
$ cd ..
$ cd -
$ cd
$ cd ~

$ pwd

$ rmdir [选项]... 目录...
