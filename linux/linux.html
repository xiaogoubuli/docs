硬件设备文件名
IDE硬盘：/dev/hd[a-d]
SCSI/SATA/USB硬盘：/dev/sd[a-p]
光驱：/dev/cdrom或/dev/sr0
软盘：/dev/fd[0-1]
打印机（25针）：/dev/lp[0-2]
打印机（USB）：/dev/usb/lp[0-15]
鼠标：/dev/mouse

修改IP：
$ vi /etc/sysconfig/network-scripts/ifcfg-xxxx
ONBOOT=yes
BOOTPROTO=static
IPADDR=192.168.1.65
NETMASK=192.168.1.255
GATEWAY=192.168.1.1
DNS1=8.8.8.8
DNS2=8.8.4.4
$ ifup xxxx
$ service network restart
$ ifconfig
$ ping www.baidu.com

配置本地iso yum源：
上传iso到指定位置
$ mount -o loop /xxx.iso /media/cdrom
$ vi /etc/yum.repos.d/CentOS-Media.repo
baseurl=file:///media/CentOS-DVD1/
        file:///media/CentOS-DVD2/
gpgcheck=0
enabled=1
$ yum install vim

安装vnc：
安装所需的组件
$ rpm -q tigervnc-server
$ yum install tigervnc-server
$ yum groupinstall Desktop
创建 VNC 用户的帐号
$ useradd pspot
$ passwd pspot

安装Oracle：
