硬件设备文件名
IDE硬盘：/dev/hd[a-d]
SCSI/SATA/USB硬盘：/dev/sd[a-p]
光驱：/dev/cdrom或/dev/sr0
软盘：/dev/fd[0-1]
打印机（25针）：/dev/lp[0-2]
打印机（USB）：/dev/usb/lp[0-15]
鼠标：/dev/mouse

分区设备文件名：
IDE硬盘：/dev/hda1
SCSI/SATA硬盘：/dev/sda1

修改IP：
$ ifconfig 设备名 IP地址 （临时生效）
示例：
$ ifconfig eth0 192.168.56.2


$ vi /etc/sysconfig/network-scripts/ifcfg-xxxx
ONBOOT=yes
BOOTPROTO=static
IPADDR=192.168.1.65
NETMASK=192.168.1.255
GATEWAY=192.168.1.1
DNS1=8.8.8.8
DNS2=8.8.4.4
$ ifup xxxx
$ service network restart
$ ifconfig
$ ping www.baidu.com

配置本地iso yum源：
上传iso到指定位置
$ mount -o loop /xxx.iso /media/cdrom
$ vi /etc/yum.repos.d/CentOS-Media.repo
baseurl=file:///media/CentOS-DVD1/
        file:///media/CentOS-DVD2/
gpgcheck=0
enabled=1
$ yum install vim

安装vnc：
安装所需的组件
$ rpm -q tigervnc-server
$ yum install tigervnc-server
$ yum groupinstall Desktop
创建 VNC 用户的帐号
$ useradd pspot
$ passwd pspot

安装Oracle：
上传zip到/usr/local/src

/etc/passwd
用户名:密码标志:用户ID:初始组ID:用户说明:家目录:登录Shell
UID 
0 超级用户（默认root）
1-499 系统用户（伪用户）
500-65535 普通用户

/etc/shadow
用户名:加密密码（!!或*表示无密码，不能登录）:密码最后一次修改日期:两次密码的修改间隔时间:密码有效期:密码修改到期前的警告天数:密码过期后的宽限天数（0或不写表示密码过期后立即失效，-1表示密码永远不失效）:账号失效时间:保留字段


/etc/group
组名:组密码标志:组ID:组中附加用户


/etc/gshadow
组名:组密码:组管理员用户名:组中附加用户

/etc/skel/
用户模板目录
