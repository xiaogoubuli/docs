插件：
JAD网址：http://varaneckas.com/jad/
将zip包解压缩即可使用。
JadClipse网址：http://sourceforge.net/projects/jadclipse/
安装：将jar包放入Eclipse的dropins文件夹中，重启Eclipse，安装完成。
配置：在Eclipse中，打开Window -> Preferences -> Java -> JadClipse， Path to decompiler填写解压后的JAD可执行程序路径，不需要加后缀，如C:\jad158g.win\jad，其他保持默认。
设置打开.class文件的默认程序：打开Window -> Preferences -> General -> Editors -> File Associator，选中*.class without source，点击Associated editors的Add按钮，选择JadClipse Class File Viewer。

取消构建项目时的JavaScript验证：
右键项目 -> Properties -> Builders | 取消勾选JavaScript Validator
